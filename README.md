# Greg's Digital Library

[![Deploy Hugo site to Pages](https://github.com/gregpuzzles1/blog_hugo/actions/workflows/hugo-pages.yml/badge.svg)](https://github.com/gregpuzzles1/blog_hugo/actions/workflows/hugo-pages.yml)

This repository is a **development/testing repo** for the actual Tech Trail Pulse website and blog at **www.techtrailpulse.com**.

It is used to experiment with site configuration, theme customization, content structure, and deployment workflows before changes are finalized for the live site.

## Tech stack

- **Hugo** for static site generation
- **Blowfish** theme for site layout and presentation
- **GitHub Pages** / GitHub Actions for deployment automation
- **CSS** for styling customizations
